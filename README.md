## ✨ 어떤 앱이에요?

유튜브에 영상 올리셨죠? 그럼 이제 **J-LIN Studio** 차례예요! 😊

한국어 제목이랑 설명만 입력하시면, AI가 알아서 척척 **50개국 언어**로 번역해드려요.
그러면 어떻게 되냐면요...

> 👉 미국 친구가 보면 → **영어**로 보여요!
> 👉 일본 친구가 보면 → **일본어**로 보여요!
> 👉 태국 친구가 보면 → **태국어**로 보여요!

이러면 전 세계 사람들이 내 영상을 더 잘 찾을 수 있어요. 와! 진짜 좋죠? 🌟

> 💚 **걱정 마세요!** 영상은 그대로 있어요. 제목이랑 설명만 다국어로 추가되는 거라서
> **진짜 빠르고 안전해요!** ✨

## 🚀 빠른 시작

### 1️⃣ 파일 받기

[GitHub Releases 페이지](https://github.com/YOUR-USERNAME/jaylin-yt-translator/releases)에서
**`J-LIN-Studio.exe`** 파일을 받으시면 돼요! 90MB 정도예요. 📥

### 2️⃣ 더블클릭으로 실행!

받은 파일을 그냥 더블클릭하면 끝이에요. 너무 쉽죠? 🎈

> 😱 **어? 이상한 파란 화면이 떠요!**
> 당황 안 하셔도 돼요~ 이건 Windows가 처음 보는 앱이라 살짝 의심하는 거예요.
> 우리 앱은 안전하니까 **"추가 정보"** 누르고 **"실행"** 버튼 클릭하시면 끝! 👍

### 3️⃣ 처음 한 번만! 설정 마법사 따라가기 🧙‍♀️

처음에만 짧은 설정 마법사가 도와드려요. 딱 **3단계만** 하시면 돼요!

| 단계 | 뭘 하나요? |
|---|---|
| ✨ 1단계 | **환영합니다** 화면 → "시작" 버튼 누르기 (쉽죠?) |
| 🔑 2단계 | Google OAuth 키 등록 ([어떻게 받아요?](#-google-oauth-발급-약-5분-걸려요)) |
| 🤖 3단계 | Gemini AI 키 등록 ([어떻게 받아요?](#-gemini-api-키-발급-약-2분이면-끝)) |

이거 한 번만 하시면, **다음부터는 그냥 쓱 메인 화면으로 들어가요!** 두 번 안 해도 돼요. 😉

## 💡 사용 방법

메인 화면 들어오시면 **4가지만 입력**하면 끝이에요! 어렵지 않아요. 💪

| 입력칸 | 어떻게 적어요? |
|---|---|
| **🎥 유튜브 영상 주소** | 영상 주소 그대로 복사해서 붙여넣기! (예: `https://youtube.com/watch?v=...`) |
| **📝 유튜브 제목** | 영상 제목을 그대로 한국어로 적어주세요 |
| **📄 유튜브 본문** | 영상 설명도 그대로! 길어도 괜찮아요 |
| **🌐 번역 언어** | 어느 나라 언어로 번역할지 골라주세요 (Tier 1 추천!) |

다 입력하셨으면 **"번역 및 자동 업로드 시작"** 버튼 콱! 누르면 끝! 🎉

> 💰 **돈 얼마 들어요?**
> 거의 안 들어요! 영상 1개에 **약 $0.0005** 들어요... 한국 돈으로 **1원도 안 돼요!** 😆
> 하루에 200개까지 무료로 쓸 수 있어요. 진짜 부담 없죠? 😎

## 🔑 Google OAuth 발급 (약 5분 걸려요)

유튜브에 자동으로 다국어 등록을 하려면, Google한테 **"이 앱 써도 돼!"** 라고 허락받아야 해요.
우리 채널만 다루는 거니까 진짜 안전해요! 👍

> 💡 **잠깐만!** 처음엔 어려워 보일 수 있는데, 천천히 따라하시면 누구나 할 수 있어요!
> 막히는 부분 있으면 잠깐 쉬었다가 다시 해도 괜찮아요~ 😊

### 발급 따라하기

**1.** [**Google Cloud Console**](https://console.cloud.google.com/) 사이트로 가요!
   평소 쓰는 Google 계정으로 로그인해주세요. 🔓

**2.** 화면 **왼쪽 위 "프로젝트 선택"** 버튼이 있어요.
   거기 눌러서 **"새 프로젝트"** 만들기! 이름은 `jlin-studio` 처럼 알아보기 쉽게 지어주세요. 😊

**3.** 왼쪽 메뉴에서 **"API 및 서비스" → "라이브러리"** 들어가요.
   검색창에 **`YouTube Data API v3`** 치고, 나오는 거 클릭 → 파란색 **"사용"** 버튼 누르기! 🎯

**4.** 다음은 **"OAuth 동의 화면"** 으로 가요.
   어렵게 들리지만 그냥 **"Google한테 이 앱 소개시켜주는 화면"** 이에요!
   - 사용자 유형: **"외부"** 선택 → 만들기
   - 앱 이름이랑 본인 이메일 입력
   - 🌟 **"테스트 사용자"에 본인 Gmail 꼭 추가하세요!** (이거 빼먹으면 안 돼요 😅)

**5.** 이제 진짜 키 만드는 단계예요! ✨
   - **"사용자 인증 정보"** 메뉴에서 **"+ 사용자 인증 정보 만들기"** → **"OAuth 클라이언트 ID"** 클릭
   - 애플리케이션 유형은 꼭 **"데스크톱 앱"** 으로! (다른 거 고르면 안 돼요!)
   - 이름 적고 **"만들기"** → 짠! 다운로드 버튼이 보여요 🎉

**6.** 다운받은 `.json` 파일을 잘 보관해주세요! 이게 우리 앱이 쓸 **열쇠** 예요. 🔑
   마법사 2단계에서 이 파일을 선택하면 돼요!

> ⚠️ **잠깐, 진짜 중요해요!**
> 이 파일은 우리 채널의 **"마스터 열쇠"** 같은 거예요.
> 다른 사람한테 절대로 보여주거나 인터넷에 올리면 **안 돼요!** 🤫
> 비밀번호처럼 본인만 알고 계세요!

## 🤖 Gemini API 키 발급 (약 2분이면 끝!)

이건 번역할 때 쓰는 **AI 비밀번호** 예요! 무료로 쓸 수 있어요. ✨

**1.** [**Google AI Studio**](https://aistudio.google.com/apikey) 들어가서 Google 계정으로 로그인! 🔓

**2.** **"Get API key"** 누르고 **"Create API key"** 클릭하면 돼요!

**3.** **`AIzaSy...`** 로 시작하는 긴 글자가 떠요.
   통째로 복사해주세요! (Ctrl+C 또는 마우스 오른쪽 클릭 → 복사) 📋

**4.** 마법사 3단계 입력칸에 붙여넣기 (Ctrl+V) → **끝!** 진짜 쉽죠? 🎉

> ⚠️ **이 키도 비밀이에요!**
> 다른 사람한테 카톡으로 보내거나 스크린샷 찍어서 보여주면 안 돼요.
> 본인만 알고 계세요! 🤐

## 📁 키들은 어디에 저장돼요?

마법사 끝나면 우리 키들이 이 폴더에 잘 숨어 있어요! 다른 사람이 못 봐요. 🏠

```
C:\Users\<내 이름>\AppData\Roaming\jaylin-yt-translator\
├── credentials.json     ← Google 키 🔑
├── token.json           ← 자동으로 만들어지는 인증 파일
├── .env                 ← Gemini AI 키 🤖
└── .quota_log.json      ← 오늘 얼마나 썼는지 기록
```

- 👫 **다른 사람과 분리됨**: 같은 컴퓨터에 다른 사용자 계정이 있어도 못 봐요. 안심!
- 🗑️ **앱 지우고 싶으시다면?**: 이 폴더도 같이 지워주시면 깔끔해져요!

## 🌐 English (brief)

**J-LIN Studio** is a Windows desktop app that auto-translates your YouTube video's
Korean title & description into up to 50 languages, registered as multi-locale metadata
via the YouTube Data API.

**Quick start**: Download `J-LIN-Studio.exe` from
[Releases](https://github.com/YOUR-USERNAME/jaylin-yt-translator/releases) → double-click →
follow the 3-step wizard (Google OAuth + Gemini API key) → paste video URL → done!

Built with PySide6 + PyInstaller. ~90 MB single .exe. Korean UI.
